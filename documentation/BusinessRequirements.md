# Business Requirements

## Project Title

Banking Executive Dashboard

---

## Background

Bank executives require a centralized dashboard to monitor financial performance, customer growth, loan portfolio, deposits, branch performance, and transaction trends.

Currently, data resides in multiple operational systems, making reporting slow and inconsistent.

---

## Objectives

Develop a Business Intelligence solution that:

- Consolidates banking data
- Provides executive KPIs
- Enables branch comparison
- Tracks loan performance
- Monitors customer growth
- Supports strategic decision-making

---

## Users

- CEO
- Executive Directors
- CFO
- Head of Retail Banking
- Branch Managers

---

## Success Criteria

- Interactive Power BI Dashboard
- Automated SQL reporting
- Fast dashboard performance
- Accurate KPIs
- Professional documentation
