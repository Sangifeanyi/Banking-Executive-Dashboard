# Banking Executive Dashboard

## Overview

This project demonstrates how SQL Server and Power BI can be used to build an Executive Banking Dashboard for senior management.

The solution includes:

- SQL Server Database
- Relational Database Design
- Stored Procedures
- Views
- Power BI Dashboard
- DAX Measures
- KPI Reporting
- Documentation

---

## Technology Stack

- SQL Server
- SQL Server Management Studio
- Power BI Desktop
- Git
- GitHub
